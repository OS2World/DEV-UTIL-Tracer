This is a version of Daniel Hildebrand's Tracer
utility, updated to work with OS/2 2.X and compiled
using emx 0.8g

You will require the emx dlls to use this executable version
of Tracer.  Alternatively, you should be able to recompile the
source using any OS/2 32bit C compiler.
